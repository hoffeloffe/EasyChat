Welcome to EasyChat

This is a demo page with the EasyChat widget loaded in file mode. Click the chat button in the bottom-right corner to try it out!

The chatbot reads from a knowledge.json file and matches your questions using smart keyword matching - no AI, no API keys, no server needed.

💡 Try asking:

• "What is EasyChat?"

• "How do I install it?"

• "Is it free?"

• "What file formats are supported?"

• "Can I use it with React?"

📁 How this works

This page includes a single <script> tag that loads the EasyChat widget. The widget fetches knowledge.json and uses keyword matching to answer questions. That's it!
